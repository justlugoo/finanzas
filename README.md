# Finanzas

App de escritorio multiplataforma para gestión financiera personal.

## Stack

- **Backend:** Rust + Tauri 2.x
- **Frontend:** Svelte 5 + CSS puro
- **Base de datos:** libSQL (Turso) con embedded replicas
- **Plataformas:** Linux (AppImage), Windows (MSI)

## Estado

En fase de Diseño completada. Próximo: Desarrollo (F0 — setup proyecto Tauri).
